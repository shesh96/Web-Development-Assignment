1) List all the tools you might use as a web developer

As a web developer, I use different tools at different stages of development. These tools help in writing code, designing websites, testing, collaboration, and deployment.

Common tools used by a web developer:

Code Editors / IDEs

Visual Studio Code

Sublime Text

IntelliJ IDEA

These tools help in writing and managing code efficiently.

Version Control

Git

GitHub / GitLab

They help track code changes, collaborate with teams, and manage project history.

Frontend Tools

HTML, CSS, JavaScript

React, Angular, or Vue

Bootstrap, Tailwind CSS

These are used to build responsive and interactive user interfaces.

Backend Tools

Node.js, Express

Python, Java

REST APIs

These tools handle server-side logic and data processing.

Databases

MySQL

MongoDB

PostgreSQL

Databases store and manage application data.

Browsers & Developer Tools

Google Chrome DevTools

Firefox Developer Tools

Used for debugging, testing, and performance analysis.

Design Tools

Figma

Adobe XD

Canva

Used to design layouts and UI before development.

Deployment & Hosting

Netlify

Vercel

AWS

Used to deploy and host websites and applications.

2) What are some of the features of PW Skills Online Lab? Explain how they are useful.

PW Skills Online Lab provides a hands-on learning environment that helps students practice real-world development skills.

Key features and their usefulness:

1. Browser-based Coding Environment
Students can write and run code directly in the browser without installing any software. This makes learning easy and quick.

2. Real-Time Practice Labs
The lab allows learners to practice coding concepts immediately after learning them, which improves understanding and retention.

3. Pre-configured Setup
All required tools and dependencies are already set up. This saves time and avoids system configuration issues.

4. Assignment-Based Learning
Students can work on assignments and projects that simulate real industry problems, helping them gain practical experience.

5. Progress Tracking
Learners can track their progress and completion status, which helps in staying consistent and motivated.

6. Beginner-Friendly Interface
The interface is simple and easy to use, making it suitable even for beginners with no prior coding experience.
